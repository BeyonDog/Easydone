发布版本

「仅构建」不会修改版本号，也不会复制到内网 Update 目录，仅供本机安装测试。

构建与发布

三按钮相互独立：仅构建只编译 NSIS 安装包；仅发布用已有产物写入内网目录；构建并发布为完整发版流程。若签名私钥有密码，请在 publish.config.json 配置 signingKeyPassword。

代码存档（GitHub）

将当前源码与最新 NSIS 安装包提交并推送到对应分支（--force-with-lease，会覆盖远程该分支历史）。请先完成「仅构建」或「构建并发布」。需本机已安装 Git 并可访问 BeyonDog/Easydone。
